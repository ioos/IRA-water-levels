## Phase Overview

| Phase | Purpose | Key Tasks / Deliverables |
|-------|---------|--------------------------|
| **Site Suggestion** | Identify a general area or specific location that would benefit from water level monitoring based on stakeholder engagement and user needs. | - Identify an area of interest<br>- Define the issue or scientific question<br>- Engage stakeholders and end users<br>- Determine whether nearby projects or existing data sources can be leveraged |
| **Planning Phase** | Evaluate the proposed site and develop a complete installation plan. | **Identify Specific Location**<br>- Desktop reconnaissance<br>- Field reconnaissance<br>- Identify installation infrastructure<br><br>**Installation Planning**<br>- Select sensor<br>- Notify property owner or steward<br>- Obtain permissions or permits<br>- Design sensor mount<br>- Obtain hardware, tools, and mounting materials<br>- Locate nearby survey benchmarks |
| **Installation** | Deploy the station and establish the reference information needed for long-term operation. | **Sensor Installation**<br>- Install sensor and housing<br>- Test communications and data delivery<br>- Install signage (if required)<br><br>**Sensor Survey**<br>- Survey sensor elevation relative to NAVD88<br>- Record sensor offset<br>- Document station information |
| **Data Pipeline / Delivery** | Configure the station for public data delivery through NERACOOS systems. | **Metadata**<br>- Complete station metadata<br>- Record contact information<br>- Document sensor specifications and site information<br><br>**QARTOD**<br>- Configure quality control tests<br>- Verify data delivery<br>- Publish dataset to ERDDAP<br>- Display station on the Mariners Dashboard |
| **Active Station** | Maintain a reliable operational station that provides high-quality, publicly available data. | - Monitor data delivery<br>- Review QARTOD flags<br>- Update metadata as needed<br>- Perform routine maintenance and surveys<br>- Support end users with reliable water level observations |
